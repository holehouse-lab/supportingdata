# About
The single file in this directory (`LS_ODM_seq_analysis-main.zip`) is a clone of the ODM-Seq analysis pipeline developed for the Langstein-Skora et al. paper. The code [is shared here on GitHub](https://github.com/gerland-group/LS_ODM_seq_analysis), but for completeness and the purpose of the Zenodo repository, we cloned and downloaded the code, which is provided here as well.

This code was obtained on Nov 30th 2025.
